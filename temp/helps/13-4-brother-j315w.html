Simple.  Followed the instructions here http://welcome.solutions.brother.com/...

  1. Download the printer driver and cupswrapper driver from ..brother.com
  2. Step 4. Install LPR driver
    • 1. Turn on the printer and connect the USB cable.
    • 2. Install LPR driver.
        sudo dpkg  -i  --force-all  <lpr-drivername>

        I needed to sudo mkdir for a lot of non existing dir's (on a new laptop)
        /usr/local/etc/cups
        /usr/local/etc/cups/certs
        /usr/local/etc/cups/ppd

        /var/spool/
        /var/spool/lpd/dcpj315w

    3. Install cupswrapper driver.

        sudo dpkg  -i  --force-all  <cupswrapper-drivername>

        /var/tmp

        It prints!

    4. Download the scanner driver and scan tool from ...brother.com/...brscan3
    5. install away!
    6. follow the instructions here to allow normal users to scan ...instruction_scn1c.html#d6

        sudo leafpad /lib/udev/rules.d/60-libsane.rules

        Add the following two lines to the end of the device list. (Before the line "# The following rule will disable ..."):

        # Brother scanner
        ATTRS{idVendor}=="04f9", MODE="0664", GROUP="scanner", ENV{libsane_matched}="yes"

     7. add user to scanner group
        sudo adduser <username> scanner

     8. To let the scan-key-tool operate it must be started with command brscan-skey
        Then Scan can be pressed on the machine.  scn3...
        add it to Xfce autostart apps or /etc/xdg/lxsession/LXDE/autostart with @brscan-skey

        Try xsane or other scanner app -might not require the scan-key-tool (I haven't tested it yet)

        Reboot.
