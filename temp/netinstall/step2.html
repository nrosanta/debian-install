Kdenlive


Updated 06-2017

Kdenlive is KDE's non-linear video editor (hence its name.. KDE-n-li-v-e) that is both powerful and easy to use. It's also well supported online.
I give some tips on it on my page, working with HD video.


Basic install

If you're not to concerned about getting the latest codecs, you can use this install line - and if you run Debian testing then kdenlive should be quite up to date.
