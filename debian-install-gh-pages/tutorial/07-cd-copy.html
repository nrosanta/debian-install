I needed to run a command to copy one cd directly to another without ripping the tracks and burning them and thus loosing audio quality.
I read here http://www.linuxtopia.org/ that this command does the job:
sudo cdrdao copy

and to get things working we need the scsi address for the reader and writer, using
cdrdao scanbus
then we make/edit the cdrdao file with 
sudo nano /etc/default/cdrdao

and add the lines with the scsi values,
write_device: "x,x,x"
read_device: "y,y,y"
except my scsi addresses were not found, I only had /dev/sr1 and /dev/sr0 reported -so I used those exactly, in the quotes " "

eject /dev/sr0
is very handy for me as my laptop CD drive is very hard to get open with its button. And now I map this in Ranger, to something like ,cd-F8

test the written cd with this command:
mplayer -cdrom-device /dev/sr1 cdda:// -cache 5000
depending on which device holds the new cd, of course!
