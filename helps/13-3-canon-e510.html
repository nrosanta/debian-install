Helps > Printers >
This printer worked first time round, it was almost too easy!

1. Get the drivers for printer and scangear http://support-in.canon-asia.com...

2. Set up the printer and power on, and connect the USB

3. Unpack the tar.gz driver archives and open a Terminal in the printer driver directory first (cnijfilter), then run

sudo ./install.sh

answer the questions and it should complete.  Print something.

If the printer doesn't register, see here http://linux.wikia.com/...

cngpijmone510 is the status monitor (also shows ink levels)


[Desktop Entry]
Name=scangear
Comment=scan a document
Exec=scangearmp
Icon=scanner
Terminal=false
Encoding=UTF-8
Type=Application
Categories=Graphics;

[Desktop Entry]
Name=E510Status
Comment=printer status
Exec=chromium http://localhost:631/printers/E510
Icon=printer
Terminal=false
Encoding=UTF-8
Type=Application
Categories=Settings;
